Vital CRM APP

概要

Vital CRM 是一款專為中小企業量身打造的雲端顧客關係管理系統。2014 年推出新版 App，設計首重於使用者在戶外跑業務時，如何迅速找到客戶、記錄拜訪重點、行事曆管理以及處理後續任務。

目標

  • 改善舊版使用套件預設的 UI 樣式導致產品沒有個性、操作不符合內容類型以及適逢各行動裝置尺寸的興起。
  • 增進主管管理、業務外出使用的便利性。
  • 引進設計流程。

角色

  • UI Designer
  • UX Designer
  • 切版

工具

  • 紙與筆
  • Illustrator
  • Sketch app
  • 4ourth Mobile Touch Template

過程

人物誌 1

  1. Susan / 34 / 經理 / 辦公室內
  2. 痛點：舊版無任務管理系統、無法掌握員工位置、沒有佈告欄的系統。
  3. 目標：立即取得資訊、知道下屬位置、指派任務、傳達重要資訊。

人物誌 2

  1. John / 25 / 業務 / 汽、機車
  2. 痛點：舊版不能一眼就觀看完客戶資料、地圖上不可以選擇客戶、在網路不穩的情況下不能使用。
  3. 目標：輕易的找到客戶、記錄與安排會議/約會、不管在哪都可以使用。

整合痛點

  1. 快速且輕易地尋找目標人物。
  2. 要比舊版更好的填寫紀錄。
  3. 新增任務系統。
  4. 新增分享系統( 地點、留言、任務/約會與會議 )。

研究

研究當時各種行動裝置尺寸、使用者操作模式以及可觸及範圍，作為日後設計版面之依據。

同一時間進行 Functional Map、UI Flow 以及 Wireframe

設計稿

  1. 既要符合現有產品的視覺規範與個性，且還要盡量遵循 iOS 與 Android Guideline。
  2. 使用者經常在外奔波，操作範圍要大且符合情境操作，例如機/汽車、單/雙手、點/握/拿...等。
  3. 資訊量龐大，不作無謂裝飾，聚焦於資訊呈現簡單易懂。
