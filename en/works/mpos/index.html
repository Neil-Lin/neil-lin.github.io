KSI mPOS

概要

KSI mPOS 是一款行動版餐飲 POS 系統，擁有多台平板分流、開立電子發票、快速點餐與商品內容客製化等功能，無論你人在哪裡，都能不受限的利用平板為顧客服務，輕鬆提升服務水準，獲取顧客好評。

目標

  • 2016 年製作新版 APP，提高餐飲業在前場有更好的使用、管理流程。
  • 快、快、快，第一線點餐要快！
  • 任何時間點都可以編輯訂單內容。
  • 快速取得會員資訊並回饋。

角色

  • UI Designer
  • UX Designer

工具

  • Sketch app
  • Zeplin
  • Flinto
  • Illustrator
  • 4ourth Mobile Touch Template

過程

  1. 什麼是 POS 系統？
  2. 舊版 POS 系統的優缺點為何？
  3. 與後台和 CRM 的關係又是為何？
  4. 有哪些硬體設備？

人物誌

  1. Grace / 17 / 飲料店前場點餐、結帳與收銀
  2. 痛點：舊版眼花撩亂、操作步驟多且不符合邏輯
  3. 目標：快、易用、好管理

觀察

  1. 在尖峰時刻點餐必須很快就處理完畢。
  2. 常常發生修改訂單的狀況。
  3. 各種優惠方案、付費方式。
  4. 查詢會員資訊。

研究

研究當時適合當餐飲業用的平板，作為日後設計版面之依據。

跨公司、跨團隊遠端溝通，一步步解析尋找舊版未被提及的痛點

因應新需求找出新操作模式，幫助使用者快速找出歷史紀錄

設計稿

  1. 使用 Zeplin 與工程師溝通並製作符合原生 APP 規範的圖片，避免無法正確傳達設計。
  2. 使用 Flinto 製作動畫，輔助工程師理解操作行為緣由與過場。
  3. 與視覺設計師合作，設計出即便是在前場混亂的情況，也要能即時掌握資訊。
