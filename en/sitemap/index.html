Website Blocks And Shortcut Keys

This website is built based on the principles of accessible web design. The main content of the website is divided into three major sections.

1. The Header Section 2.The Main Content 3.The Footer Section

The Accesskey settings for this website are as follows:：

Alt+U：header
, main menu, including switch language&copyright
Alt+C：main content
Alt+Z：footer, including switch language&copyright

If you are using Firefox as your browser, the method for using the Accesskeys is Alt+Shift+shortcut. For example, Alt+Shift+C will navigate to the main content of the webpage, and so on.

Additionally, if you are a Mac user, the method for using the Accesskeys is Control+Option+shortcut.If none of the above methods work, it is recommended to check if there are any custom keybindings or settings that may be affecting the use of Accesskeys.

※ When the project tabs on this website cannot be clicked using a mouse, you can navigate the information using the following keyboard shortcuts.

← → or ↑ ↓: Move the tab order by pressing the left/right or up/down arrow keys.

Tab: After hovering over the tab, you can use the Tab key to jump to the content and browse the data. When encountering radio buttons, please use the ← → or ↑ ↓ keys to move through the options in the desired order.

Shift + Tab: Pressing Shift + Tab allows you to navigate back to the previous data entry. When you return to the tab items, you can continue using the ← → or ↑ ↓ keys to move through the tab order.
